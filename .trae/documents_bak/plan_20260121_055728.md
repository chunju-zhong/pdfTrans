## PDF翻译工具文本显示问题修复计划

### 问题分析
翻译生成的文档没有显示任何翻译文本，主要原因是：
1. 系统字体选择错误：选择了不支持目标语言的Symbol字体
2. 字体语言支持检测不准确：误判Symbol字体支持目标语言
3. 文本溢出问题：由于字体错误导致宽度计算错误
4. 字体遍历顺序问题：先检查到了不适合的字体
5. Symbol字体的特殊性：能被加载但不支持常规文本显示

### 修复目标
- 确保选择的字体真正支持目标语言
- 过滤掉不适合的特殊字体（如Symbol）
- 改进通用字体选择逻辑，适合所有语言
- 增强文本溢出处理，减少文本块绘制失败
- 添加可靠的字体回退机制

### 修复方案

#### 1. 优化字体语言支持检测逻辑
**修改点**：改进`_check_font_support`方法，使用更准确的检测方式
**实现思路**：
```python
def _check_font_support(self, font_name, target_lang):
    """检查字体是否真正支持目标语言
    
    Args:
        font_name (str): 字体名称
        target_lang (str): 目标语言代码
        
    Returns:
        bool: 字体是否真正支持目标语言
    """
    # 1. 首先过滤已知的特殊字体（适用于所有语言）
    special_fonts = ['Symbol', 'Wingdings', 'Webdings', 'ZapfDingbats', 'Dialog', 'DialogInput', 'Monospaced']
    if font_name in special_fonts:
        logger.debug(f"字体 '{font_name}' 是特殊字体，跳过检测")
        return False
    
    try:
        # 2. 尝试加载字体
        font = ImageFont.truetype(font_name, 12)
        
        # 3. 获取目标语言的测试字符
        test_chars = {
            'zh': '你好世界',
            'en': 'Hello World',
            'ja': 'こんにちは世界',
            'ko': '안녕하세요 세계'
        }
        test_char = test_chars.get(target_lang, test_chars['en'])
        
        # 4. 检查字体是否包含测试字符的字形
        # 使用getbbox检查每个字符的宽度，替换字符宽度通常为0或固定值
        for char in test_char:
            bbox = font.getbbox(char)
            char_width = bbox[2] - bbox[0]
            # 如果字符宽度为0或非常小，说明字体不支持该字符
            if char_width < 1:
                logger.debug(f"字体 '{font_name}' 不包含字符 '{char}' 的字形")
                return False
        
        logger.debug(f"字体 '{font_name}' 支持目标语言 '{target_lang}'")
        return True
    except Exception as e:
        logger.debug(f"字体 '{font_name}' 不支持目标语言 '{target_lang}': {e}")
        return False
```

#### 5. 优化系统字体获取
**修改点**：改进`_get_system_fonts`方法，优化字体质量
**实现思路**：
```python
def _get_system_fonts(self):
    """从系统中获取可用字体列表
    
    Returns:
        list: 系统可用字体名称列表
    """
    system_fonts = []
    try:
        # 获取系统字体目录
        if sys.platform == 'win32':
            font_dirs = [r'C:\Windows\Fonts']
        elif sys.platform == 'darwin':
            font_dirs = [
                '/System/Library/Fonts',
                '/Library/Fonts',
                os.path.expanduser('~/Library/Fonts')
            ]
        else:
            font_dirs = [
                '/usr/share/fonts',
                '/usr/local/share/fonts',
                os.path.expanduser('~/.fonts')
            ]
        
        # 遍历字体目录，获取字体文件
        for font_dir in font_dirs:
            if os.path.exists(font_dir):
                for root, _, files in os.walk(font_dir):
                    for file in files:
                        if file.endswith(('.ttf', '.otf')):  # 只考虑TrueType和OpenType字体，忽略.ttc集合字体
                            # 获取字体名称（不含扩展名）
                            font_name = os.path.splitext(file)[0]
                            system_fonts.append(font_name)
        
        # 去重并排序
        system_fonts = list(set(system_fonts))
        system_fonts.sort()  # 按字母顺序排序，提高可预测性
        logger.info(f"从系统获取到 {len(system_fonts)} 种可用字体")
    except Exception as e:
        logger.warning(f"获取系统字体列表失败: {e}")
        system_fonts = []
    
    return system_fonts
```
