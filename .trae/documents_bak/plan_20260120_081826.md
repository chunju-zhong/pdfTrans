## 为 `pdf_extractor.py` 添加详细日志功能（完整文本输出）

### 修改内容

1. **添加日志导入和初始化**
   - 在文件头部添加 `import logging`
   - 创建模块级 logger：`logger = logging.getLogger(__name__)`

2. **在关键位置添加日志输出（完整文本内容）**
   - **开始提取时**：记录文件路径、总页数
   - **提取文本块时**：记录每个文本块的**完整内容**、位置、字体、大小、颜色等属性
   - **提取表格时**：记录表格位置（页码、索引）和内容摘要
   - **完成提取时**：记录提取汇总信息（总文本块数、总表格数）

### 日志输出示例

```
2026-01-20 10:30:00 - modules.pdf_extractor - INFO - 开始提取PDF: example.pdf
2026-01-20 10:30:00 - modules.pdf_extractor - INFO - PDF总页数: 5
2026-01-20 10:30:00 - modules.pdf_extractor - INFO - 第1页: 提取到10个文本块
2026-01-20 10:30:00 - modules.pdf_extractor - INFO - 文本块: "智能体与智能体架构导论" (位置: x0=100.0, y0=50.0, x1=400.0, y1=80.0, 字体=Arial-BoldMT, 大小=18.0, 粗体=True)
2026-01-20 10:30:00 - modules.pdf_extractor - INFO - 表格: 第1页-表格0, 包含5行3列
2026-01-20 10:30:00 - modules.pdf_extractor - INFO - 提取完成: 总文本块=50, 总表格=3
```

### 不修改的内容

- 保持现有的 `extract_text()`、`extract_page_text()`、`extract_tables()`、`get_metadata()` 方法逻辑不变
- 保持返回数据结构不变
- 只添加日志记录功能
