# 开发计划：PDF指定页翻译功能

## 1. 创建feature分支

* 从develop分支创建feature/pdf-page-translation分支

## 2. 前端UI修改

### 2.1 添加页码选择功能

* 在index.html中添加页码选择区域，包括：

  * 当前PDF总页数显示

  * 页码输入框，支持单个页码和页码范围（如"1-5,7,9-10"）

  * 全选/取消全选选项

### 2.2 实现文件选择后自动获取PDF页数

* 在main.js中添加文件选择事件监听

* 实现获取PDF页数的API接口调用

* 更新UI显示总页数

## 3. 后端功能修改

### 3.1 添加获取PDF页数的API接口

* 在app.py中添加获取PDF页数的路由

* 实现PDF页数提取逻辑

### 3.2 修改翻译路由支持页码范围参数

* 在translate路由中添加页码范围参数接收

* 传递页码范围参数给翻译服务

### 3.3 修改翻译服务支持指定页码范围

* 修改translation\_service.py中的process\_translation方法

* 添加页码范围解析逻辑

* 只处理指定页码的文本块

## 4. 测试

* 添加测试用例验证页码范围解析

* 测试单页翻译功能

* 测试多页范围翻译功能

* 测试全选功能
