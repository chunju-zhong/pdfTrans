### 失败的测试用例

1. **TestPdfPageTranslationIntegration.test_process_translation_with_page_range**
   - 错误：`'dict' object has no attribute 'block_text'`
   - 原因：测试用例创建了字典格式的 `merged_blocks`，但代码期望 `MergedBlock` 对象

2. **TestSemanticMerge.test_merge_semantic_blocks**
   - 错误：`TypeError: argument of type 'MergedBlock' is not iterable`
   - 原因：测试用例使用 `'max_width' in merged_block` 检查属性，但 `merged_block` 现在是 `MergedBlock` 对象，不是字典

3. **TestSemanticMergeExtended.test_merge_consecutive_body_blocks**
   - 错误：`TypeError: 'MergedBlock' object is not subscriptable`
   - 原因：测试用例使用 `merged_block['original_blocks']` 访问属性，但 `merged_block` 现在是 `MergedBlock` 对象，不是字典

4. **TestSemanticMergeExtended.test_merge_multiple_sequential_blocks**
   - 错误：`TypeError: 'MergedBlock' object is not subscriptable`
   - 原因：同上，测试用例使用字典访问语法访问 `MergedBlock` 对象

5. **TestSemanticMergeExtended.test_single_block**
   - 错误：`TypeError: 'MergedBlock' object is not subscriptable`
   - 原因：同上，测试用例使用字典访问语法访问 `MergedBlock` 对象
