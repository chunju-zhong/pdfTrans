## AI Ping翻译API实现修改计划

### 问题分析
当前aiping_translator.py的实现与AI Ping API文档存在多处不符：
1. **API调用方式错误**：当前使用简单的translate端点，而AI Ping使用类似OpenAI的chat/completions方式
2. **缺少模型指定**：没有指定模型名称，而文档要求必须提供model参数
3. **请求格式错误**：当前请求体包含text、source_lang、target_lang，而AI Ping需要构造messages数组
4. **响应解析错误**：当前尝试获取translated_text字段，而AI Ping响应格式是choices数组
5. **缺少依赖**：requirements.txt中缺少openai库

### 配置现状
- .env文件中已经正确配置了AIPING_API_KEY和AIPING_API_URL
- API_URL已经配置为https://aiping.cn/api/v1（正确）

### 修改方案

#### 1. 更新依赖
- 在requirements.txt中添加openai库

#### 2. 修改aiping_translator.py
- 导入openai库，使用OpenAI SDK调用AI Ping API
- 构造符合AI Ping要求的messages格式
- 添加model参数指定（默认使用DeepSeek-R1-0528）
- 使用chat completions方式调用
- 正确解析响应格式
- 保持与Translator基类的接口兼容性

#### 3. 修改内容

**aiping_translator.py修改**：
```python
import os
from openai import OpenAI
from .translator import Translator

class AipingTranslator(Translator):
    """aiping翻译API实现
    
    继承自Translator基类，实现aiping翻译API的调用逻辑。
    """
    
    def __init__(self, api_key, api_url=None, model="DeepSeek-R1-0528"):
        """初始化aiping翻译器
        
        Args:
            api_key (str): aiping翻译API的密钥
            api_url (str, optional): aiping翻译API的请求地址
            model (str, optional): 要使用的模型名称
        """
        super().__init__(api_key, api_url)
        self.api_url = api_url or "https://aiping.cn/api/v1"
        self.model = model
        # 初始化OpenAI客户端
        self.client = OpenAI(
            base_url=self.api_url,
            api_key=self.api_key,
        )
    
    def translate(self, text, source_lang, target_lang):
        """使用aiping翻译API翻译文本
        
        Args:
            text (str): 要翻译的文本
            source_lang (str): 源语言代码
            target_lang (str): 目标语言代码
            
        Returns:
            str: 翻译后的文本
        """
        # 验证语言代码
        if not self._validate_language(source_lang) or not self._validate_language(target_lang):
            raise ValueError(f"不支持的语言代码: {source_lang} 或 {target_lang}")
        
        # 文本预处理
        processed_text = self._preprocess_text(text)
        
        try:
            # 调用AI Ping API
            response = self.client.chat.completions.create(
                model=self.model,
                messages=[
                    {
                        "role": "system",
                        "content": f"你是一个专业的翻译助手，请将{source_lang}文本翻译成{target_lang}，保持原文意思和格式。"
                    },
                    {
                        "role": "user",
                        "content": processed_text
                    }
                ]
            )
            
            # 解析响应
            translated_text = response.choices[0].message.content
            
            # 文本后处理
            return self._postprocess_text(translated_text)
                
        except Exception as e:
            raise Exception(f"aiping翻译API请求失败: {str(e)}")
```
