# 章节Markdown生成问题修复 - 验证清单

- [ ] 检查章节识别逻辑是否正确检测到所有章节，包括嵌套章节
- [ ] 检查章节映射逻辑是否为所有页码正确关联对应的章节
- [ ] 检查章节内容组织逻辑是否为所有章节创建条目
- [ ] 检查章节内容过滤逻辑是否正确处理嵌套章节
- [ ] 检查章节内容处理逻辑是否正确处理文本块
- [ ] 检查章节内容处理逻辑是否正确处理图像
- [ ] 检查章节内容处理逻辑是否正确处理表格
- [ ] 验证生成的章节文件数量与章节数量一致
- [ ] 验证每个章节文件包含完整的内容
- [ ] 验证嵌套章节正确生成，文件名包含正确的层级编号
- [ ] 验证章节索引文件正确生成，包含所有章节的链接
- [ ] 验证修复不影响其他功能的正常运行
